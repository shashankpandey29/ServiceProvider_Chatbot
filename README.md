# ServiceProvider_Chatbot
Home Service Chatbot using HTML, CSS, JavaScript, and PHP with Google Gemini AI. Runs on WAMP server to answer user queries about cleaning, plumbing, and electrical services. Provides smart, real-time assistance with an easy chat interface and responsive design.

->  Home Service Chatbot (Gemini AI + PHP)

This is a web-based **Home Service Chatbot** built using **HTML, CSS, JavaScript, and PHP**, powered by **Google Gemini AI**.  
It helps users with queries related to home services such as cleaning, plumbing, electrical repair, and pest control.  
The chatbot runs on a **WAMP server** and communicates with the Gemini API for smart, real-time responses.


-> Features
- Gemini AI-powered chatbot
- Real-time chat interface
- Built using HTML, CSS, JavaScript, and PHP
- Runs locally on WAMP server
- Handles queries for home services like cleaning, plumbing, and repairs
- Simple, responsive design


->  Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Server:** WAMP  
- **AI Engine:** Google Gemini API
